🍓 Strawberry Marketplace
Strawberry Marketplace is an educational multi-vendor platform developed as a team project. The system includes roles for admin, vendor, and customer. Users can place orders and leave product reviews.


User Roles
Admin – manages the platform via the admin panel.

Vendor – can register, create a store, and manage their products.

Customer – can browse products, add to cart, place orders, and leave reviews.


Features
User registration and login

Product and store management for vendors

Product browsing and order placement for customers

Review and rating system
